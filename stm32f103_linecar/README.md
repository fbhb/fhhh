# STM32F103 Line Following Car

## Compile / Flash / Tuning Instructions
- Ensure you have Keil MDK-ARM installed.
- Open the project file `stm32f103_linecar.uvprojx` in Keil.
- Select the appropriate target and compile the project.
- Flash the binary to your STM32F103 using the debugger or ST-Link.

## First-Run Checklist
1. Ensure all wiring is correct as per the provided wiring table.
2. Power on the circuit and verify the voltage levels.
3. Test motor controls manually before running the line-following algorithm.
